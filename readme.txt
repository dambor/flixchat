# FlixChat

FlixChat is an interactive chat-based Netflix recommendation application that helps users discover content tailored to their preferences.

## Features

- Natural language chat interface for expressing viewing preferences
- Personalized content recommendations based on viewing history
- Smart content filtering by genre, mood, and time constraints
- Detailed content information with expandable cards
- Mobile-responsive design

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This application is optimized for deployment on Vercel. Simply push to a GitHub repository and connect to Vercel for automated deployments.

```bash
# Build for production
npm run build
# or
yarn build
```

## How to Use

1. Connect your Netflix account (simulated in this demo)
2. Chat naturally about what you want to watch
3. Browse the personalized recommendations
4. Click on a movie for more details

## Technologies Used

- Next.js
